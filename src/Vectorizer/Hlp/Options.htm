Vectorization Options

General

Option Batch command Description

Transform Method
Center lines /cl Vectorized line is placed in the center of raster line. It's default option.
Inside contour /ic Vectorized line is placed at the inside contour of raster object.
Outside contour /oc Vectorized line is placed at the outside contour of raster object.
Line merging
Merge factor /mf=floatValue

Enables to merge lines into one. The merge factor defines how big can be gaps between line segments comparatively its length. If you want to disable line merging set zero value.

Before merging

After merging
Merge deflection /maxmd=floatValue Defines maximal deflection for merging.
Maximal gap length /maxgl=intValue Merges lines if the gap length is less than the selected value.

Vectorization
Maximal deflection /maxd=floatValue Defines maximal deflection of vectorized lines from raster ones.
Color detection  /cd Enables color-sensitive vectorization process. Otherwise the picture is considered as black and white.

Filter

Max spot area /maxsa=intValue Doesn't vectorize raster objects with area (number of pixels) less than the value.
Min polyline length  /minpl=intValue Excludes polylines with length less than the selected value.
Max polyline length /maxpl=intValue Excludes polylines with length more than the selected value.

floatValue - a value with float point;
intValue - an integer value.

All measure units are pixels.
