Batch mode

You may work with the program by using the command string:

GMVectorizer.exe sourceImageFile [options]

sourceImageFile - an image file, the supported graphics formats are: bmp, gif, jpeg, tiff, png, icon, wmf, emf.

See the list of possible options in the topic Vectorization Options. Besides it includes:

/out=outputVectorFile - defines the file name to store the results of the vectorization process, extention of the file should be one of: dxf, svg, vml, htm.

You may create one batch file to vectorize a number of files with individual options at once.

Sample

VectorizeAll.bat

GMVectorizer Raster1.bmp /out=VectorizedRaster1.dxf
GMVectorizer Raster2.gif /out=VectorizedRaster2.svg /maxd=2.5 /cd
