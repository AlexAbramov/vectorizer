Installation

Geomethod Vectorizer setup program unpacks the necessary files and shortcuts on your computer. This program is distributed for free and you are licensed to use this software without any charge.

Files

GMVectorizer.exe - Geomethod Vectorizer program
GMVectorizer.chm - html help file for the program

Uninstall

To uninstall the program click 'Add/Remove Programs' item at Control Panel, then choose 'Geomethod Vectorizer' item and click 'Remove' button.
