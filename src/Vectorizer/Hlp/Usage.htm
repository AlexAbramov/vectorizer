Usage

GMVectorizer can work as GUI application, command line tool or Automation server.

1. GUI application.

To vectorize an image open it with GMVectorizer program, click Vectorize button or Vectorization/Start menu item and save the document with desired vector format.

2. Command line tool. 

You may create simple batch file and launch it from command line to vectorize a number of images with individual settings. See Batch mode for details.

3. Automation server.

The vectorizer functionality can be used by other programs via Automation. See ProgrammingReference for details.
