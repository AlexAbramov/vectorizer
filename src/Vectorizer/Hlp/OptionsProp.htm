Document Object

Options

String like "/maxd=1.5 /cd" containing information about vectorization Options. See Batch mode for details.

Syntax VC

BSTR Options

Syntax VB

Options As String

Sample

Private Sub Vectorize_Click()
vectorizerDoc.Open (App.Path + "\Sample.bmp")
vectorizerDoc.Options = "/maxd=2.5 /cd"
vectorizerDoc.Vectorize
vectorizerDoc.Save (App.Path + "\VectorizedSample.dxf")
End Sub
