Menu commands and buttons

Menu command

 Button

Description
File    
   Open Opens a graphics file (bmp, gif, jpeg, tiff, png, icon, wmf, emf).
   Save Saves vector data into a file (dxf, svg, vml, html).
   Save As Batch   Saves current configuration as batch file. See also Batch mode.
   Exit   Quites the application.
View    
Zoom In Magnifies image.
Zoom Out Zooms out image.
Fit image to window   Displays the whole image in the window. 
Raster Shows raster in the view.
Vector Shows vector data in the view.
Vectorization    
   Start Vectorizes the opened graphics file.
   Options Shows dialog with options for the vectorization process.
Help    
   Help   Shows the html help.
   Register   Information how to register the program to have fully functional version (Registration).
   About Shows About box with information about the program version and copyright.
  Context help. Display help for clicked on buttons, menus and windows.
