Registration

Initially the downloaded program is non-registered and works as evaluation version with some restrictions. It can save limited number of vectorized polylines. Thus you can evaluate this program. If you wish to have fully functional version without limitations you should register the program.

To register the program click the menu item 'Help/Register' and follow the instructions in the appeared dialog. After  you have registered the program you receive a key. Put this key into the field 'Unlock code' and press Ok button. Restart the program. The menu item 'Help/Register' should be disabled. It means that your program is registered and works in fully functional mode.

Please keep in mind that your registered program is bound to your instance of Operation System. If you try to use it with another instance of the operation system the program will work as evaluation version with the restrictions mentioned above.
